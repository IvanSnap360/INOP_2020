Основание изготавливается из пластика, фанеры или иного листового материала толщиной 5 мм.

Файлы:
- base.dxf - макет для лазерной резки или фрезерования основания полигона (содержит все размеры)
- base.pdf - чертеж с размерами элементов полигона
- Senior_ManIRS - макет покрытия, может быть распечатан на баннерной ткани или самоклейке и нанесен поверх основания. При нанесении дополнительных логотипов учитывайте, что в номинации могут активно использоваться камеры для распознавания цветов, форм и размеров объектов, дополнительные логотипы не должны мешать участникам.
- каталог LEGO instruction - инструкции по сборке игровых элементов из деталей LEGO
